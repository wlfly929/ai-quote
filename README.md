# ai-
AI 灵感名言 · Vue3 全栈实践  基于 Vue3 + TypeScript + Vite + DeepSeek API 构建的 AI 名言生成工具。  · 核心交互：调用 DeepSeek 大模型实时生成名言，通过 SSE 流式输出实现逐字展示打字机效果，优化 AI 交互体验。 · 工程化实践：Vite 环境变量管理 API Key，TypeScript 类型安全，QuoteCard 子组件拆分，props/emit 父子通信。 · 数据持久化：localStorage + watch 自动存储收藏，刷新不丢失。 · 毛玻璃 UI 设计，适配移动端与桌面端。  技术栈：Vue3 · TypeScript · Vite · DeepSeek API · localStorage
